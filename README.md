# 📈 Ray 投資中心（investment-hub）

手機看盤入口（用 GitHub App 開啟本 repo）：

- **今日簡報** → [`briefings/`](briefings/) 資料夾，檔名 `YYYY-MM-DD.md`，每天 08:00 自動更新
- **自選股清單** → [`watchlist.md`](watchlist.md)，手機可直接編輯（點鉛筆 ✏️ → 改完 Commit changes），隔天簡報自動生效
- 電腦版即時儀表板在 Cowork 側欄（Artifacts → Watchlist Dashboard）

> 全系統零費用。內容為研究參考，非投資建議。
